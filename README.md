# memory_card_game

A fun and interactive Memory Card Matching Game built using HTML, CSS, and JavaScript. This game helps improve memory and concentration by matching card pairs.

🎮 Features
Multiple difficulty levels (Easy, Medium, Hard)
Smooth card flip animations
Clean and simple UI
Move counter
Restart / Replay option
🚀 How to Play
Choose a difficulty level
Click on a card to flip it
Match two cards with the same design
If they match, they stay open
If not, they flip back
Match all cards to win 🎉
🛠️ Technologies Used
HTML
CSS
JavaScript
▶️ How to Run

Open index.html in any browser.
